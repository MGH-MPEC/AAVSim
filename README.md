# AAVSim Microsimulation Model

The **A**NCA-**A**ssociated **V**asculitis **Sim**ulation (AAVSim) model is a computer-based, state-transition, decision tree model of the progression and outcomes of ANCA-Associated Vasculitis disease and treatment. 

### Requirements

Requires C++ and C++ compiler (MinGW used for this project).

### Compilation Instructions
Use the following command in the command line/terminal:

>mingw64-make Makefile

Minor changes may have to be made to the Makefile depending on whether you are compiling on a Windows, Mac, or Linux machine. See lines 32-39 in the Makefile to make adjustments as needed.
